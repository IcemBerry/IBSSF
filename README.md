# IBSSF (IcemBerry's Simple Spring FrameWork)
**做什么？** 基于传统SSM & Apache Shiro的简单整合框架，用于快速开发，前端使用简单便捷的Bootstrap做页面展示。

### 版本信息
- Spring Framework `4.3.14.RELEASE`
- MyBatis `3.4.5`
- Apache Shiro `1.4.0`
- JDK `1.8`
- Bootstrap `4.1.0`

- Apache Shiro `已完成未测试`
